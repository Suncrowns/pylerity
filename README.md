# Pylerity

Python SDK for Celerity panel (https://github.com/ClickDevTech/CELERITY-panel). Work for 1.3.1 version, requiring python 3.12+. 

## 🚀 Project abilities

- **View server statistic**
- **Add and get users from panel**
- **Auth with api key**

## 📋 Requierments

- Python 3.12+
- CELERITY Panel 

## 📦 Installation

```bash
git clone https://github.com/Suncrowns/pylerity.git
cd pylerity
pip install -r requirements.txt
pip install -e .
```
